1 Sum of Two Numbers

Write a function called sum that takes two numbers as arguments and returns their sum.

Test Cases:

sum(2, 3) should return 5

sum(-1, 5) should return 4

sum(0, 0) should return 0

Remember that adding two numbers in JavaScript is done with the + operator. Make sure to return the result instead of just logging it.

2 Even or Odd

Write a function called isEven that takes a number as argument and returns true if the number is even, and false if it's odd.

Test Cases:

isEven(4) should return true

isEven(7) should return false

isEven(0) should return true

You can use the modulus operator (%) to check if a number is even. If a number modulo 2 equals 0, then it's even.

3 Reverse a String

Write a function called reverseString that takes a string as argument and returns the reversed string.

Test Cases:

reverseString("hello") should return "olleh"

reverseString("JavaScript") should return "tpircSavaJ"

reverseString("") should return ""

There are several ways to reverse a string in JavaScript. You can convert the string to an array, reverse it, and then join it back to a string.

4 Find Maximum Number

Write a function called findMax that takes an array of numbers and returns the largest number in the array.

Test Cases:

findMax([1, 5, 3, 9, 2]) should return 9

findMax([-1, -5, -3]) should return -1

findMax([10]) should return 10

You can use a loop to go through each element in the array and keep track of the maximum value. Alternatively, you can use the Math.max function with the spread operator.

5 Factorial Calculator

Write a function called factorial that takes a number and returns its factorial.

Test Cases:

factorial(5) should return 120

factorial(0) should return 1

factorial(3) should return 6

The factorial of a number is the product of all positive integers less than or equal to that number. Remember that 0! is 1. You can use a loop or recursion to solve this problem.
